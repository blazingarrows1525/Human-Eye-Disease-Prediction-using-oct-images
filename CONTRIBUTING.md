# Contributing

Thanks for your interest in contributing to this project! Please follow these simple guidelines.

- Fork the repository and create your branch from `main`.
- Write clear, focused commits with descriptive messages.
- Keep changes small and test locally before submitting a pull request.
- Open a pull request describing the change, why it's needed, and any testing you've done.

We welcome bug fixes, documentation improvements, and small features. For large changes, open an issue first to discuss the design.
